# Project Setup Guide

## 1. Creating a virtual environment

```bash
python -m venv .venv
```

## 2. Activating the virtual environment

```bash
source .venv/bin/activate
```

```windows
venv\Scripts\activate.bat
```

## 3. Installing dependencies

```bash
pip install -r requirements.txt
```

## 4. Running the application

```bash
python app.py
```
